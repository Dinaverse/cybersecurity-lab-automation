# Python Security Analytics

## Description
Ce dépôt contient une collection de 40+ scripts Python développés pour l'automatisation de la cybersécurité, l'analyse de données système et la recherche.

## Organisation du code
- **/security** : Scripts d'analyse de sécurité (logs, réseaux, accès).
- **/math_and_data** : Manipulation mathématique et analyse de données (NumPy).
- **/programming_exercises** : Exercices de manipulation de structures de données.

## Inventaire des scripts (Résumé)
| Catégorie | Description des fonctionnalités |
| :--- | :--- |
| **Security** | Analyse de logs (`failed_login.py`), reconnaissance réseau (`analyse_paquets.py`), gestion des accès (`filtrage_utilisateurs.py`). |
| **Math & Data** | Calculs numériques, factorielles, manipulation de bibliothèques mathématiques. |
| **Programming** | Scripts de révision (POO, boucles, gestion de listes, fichiers). |

## Usage
Tous les scripts sont conçus pour être utilisés de manière autonome via Python 3. Aucune dépendance externe complexe n'est requise en dehors des bibliothèques standards ou `numpy` pour les scripts de la catégorie *math_and_data*.
