# Cybersecurity Lab Automation

## Description
A suite of autonomous agents and security analytics scripts for Kali Linux. This repository automates log monitoring, network reconnaissance, and security analysis.

## Content
- **Agent Scripts:** Security-Ops (Log Monitoring), Net-Analyzer (Hourly Recon), R&D Agent.
- **Security Analytics:** Python-based scripts for log parsing, network scanning, and access control.
- **MCP Bridge:** Secure integration layer for security tools.

## Technical Documentation
- [MCP Security Fix](docs/MCP_SECURITY_FIX.md)

## Usage
The agents are designed to run as background services within the laboratory infrastructure.
