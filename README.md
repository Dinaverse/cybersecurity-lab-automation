# 🛡️ Python Security Suite: Scanners & Testing

A collection of custom Python-based security tools designed for vulnerability scanning, penetration testing, and secure system analysis.

## 🛠️ Tools Included

### 🔍 Vulnerability Scanners
- `scanner.py`, `scanner1.py`, `scanner2.py`, `scanner3.py`: A series of evolving scanners designed for various network and application-level vulnerability assessments. These tools automate the identification of open ports, services, and common misconfigurations.

### 🧪 Injection Testing
- `test_ssti.py`: A specialized utility for testing Server-Side Template Injection (SSTI) vulnerabilities, providing automated payload generation and response verification.

## 🏗️ Usage
These scripts are intended for use in controlled, authorized environments only.

```bash
# Example usage (SSTI Test)
python3 test_ssti.py --url http://target-app.local
```
